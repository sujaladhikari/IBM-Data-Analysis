
## About this Project
In this course I have applied various Data Analytics skills and techniques that I have learned as part of the previous courses in the IBM Data Analyst Professional Certificate. I am assuming the role of an Associate Data Analyst who has recently joined the organization and be presented with a business challenge that requires data analysis to be performed on real-world datasets.
